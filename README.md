## What is RWManager?

A simple and powerful management script for **installing, removing, and managing the RemnaWave(https://docs.rw/docs/install/remnawave-panel/) panel** (a proxy management panel based on Xray-core).

## How to Run?

Run the following script on your server (Debian or Ubuntu):
```
bash <(curl -Ls https://raw.githubusercontent.com/Erfan-XRay/RWManager/main/RemnaWave.sh)
```
## How to Use?

After running the script, an interactive colorful menu will appear, allowing you to easily install the RemnaWave panel, uninstall it, update, or manage configurations.

(The menu is intuitive and user-friendly; no specific tutorial video is available yet.)

## ⚙️ Features

- Quick and easy installation of the RemnaWave panel
- Complete uninstallation of the panel and related files
- Interactive CLI interface with colorful and beautiful menu
- Support for Debian and Ubuntu systems
- Simple management without needing advanced Docker knowledge or manual configurations
- Compatible with advanced Xray protocols (such as VLESS with XHTTP and Reality)
